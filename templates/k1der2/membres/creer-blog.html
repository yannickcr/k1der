Créer un blog

Cette page vous permet de vous créer un blog sur K1der.net

L'adresse de votre blog sera alors http://blog.k1der.net/{pseudo}

Description des blogs :

  • Blogs propulsés par DotClear 2
  • Nombreux plugins pré-installés
  • Nombreux thèmes pré-définits
  • Possibilité de créer son propre thème
  • Espace pour le stockage de vos images
  • etc.
